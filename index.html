О проекте

KAZBT.COM отслеживает доступность в Казахстане более 1,000,000 сайтов в Интернете.

Счётчик недоступных сайтов обновляется каждый день. Каждую неделю производится проверка доступности. В скором времени можно будеть просмотреть исторические данные для каждого сайта (неделя, месяц, год).

Блокировки сайтов в Казахстане могут производиться только по решению суда. Однако, очень часто отсутствует прозрачность в деятельности исполнительных органов. Например, неясно почему недоступны такие сайты как ██████, ██████████ или █████████.

Как это работает

Чтобы проверить доступность сайта, выполняется два HTTP HEAD запроса с казахстанского и заграничного IP. Сайт является заблокированным только в том случае, если сервер доступен за пределами РК, но недоступен с казахстанского IP.

Имейте ввиду, что результат проверки сайта зависит от доменного адреса. Например: www.example.com может быть доступен, а example.com — недоступен (другой сервер).

Disclaimer

Это совершенно безопасный тест и при проверке вашего сайта вашему серверу ничего не будет.

KAZBT.COM делает только простой HTTP HEAD запрос и избегает создания нежелательного трафика. Результат проверки отдельного сайта кэшируется на 3 часа.
