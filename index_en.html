About

KAZBT.COM tracks whether 1M+ most-visited websites in the world are accessible in Kazakhstan.

A website can be blocked in Kazakhstan only by a court order. However, often, there is a real lack of transparency. For example, it in unclear why some websites like ██████, ██████████, or █████████ are still blocked.

How it works

Two HTTP requests to a website are made from kazakhstani and non-kazakhstani IP addresses. A website is reported as blocked only when the server was inaccessible from an IP in Kazakhstan, but acessible from outside the country.

Please note that query results depend on domain names. For example, www.example.com can be reported as accessible, but example.com as inaccessible (diffrent host IP).

Statistics

All query results are cached for 3 hours. The blocked websites counter is updated every day.

The status of 1M websites is checked every week. Soon anyone will be able to see the historical data for all websites.
